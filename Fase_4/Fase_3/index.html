Propósito del Sitio

Esta pagina esta pensada como un recurso para conocer cuales son las caracteristicas básicas de CSS en su sintaxix y de como aplicarlas a un documento HTML con sus diferentes selectores y recursos. Tambien veremos los Frameworks más usados de CSS y sus aplicaciones.

Fundamentos de CSS (Cascading Style Sheets)

Es un lenguaje de programación muy parecido a HTML que permite aplicar estilos a los distintos elementos de las páginas web, de modo que los títulos, listas y párrafos pueden verse igual en todas y cada una de las páginas.

Sintáxix Básica de CSS

Cada estilo CSS se compone de tres partes:
  1. Un selector
  2. Una propiedad
  3. Un valor

Selector

Indica a qué etiqueta HTML se aplicará el estilo.En este caso estamos seleccionando el body de la pagina

body { }

Propiedad

Indica a qué propiedad de la etiqueta se aplicará el estilo, por ejemplo al color de fondo, al tamaño de la letra, etc.En este ejemplo estamos cambiando el color de fondo del body.

body { background-color : }

Valor

Indica el valor de la propiedad estamos afectando. En el ejemplo apreciamos que le estamos asignando el color #FF0000

body { background-color : #FF0000; }

Frameworks de CSS

Bootstrap es un framework CSS desarrollado por Twitter en 2010, para estandarizar las herramientas de la compañía. Inicialmente, se llamó Twitter Blueprint y un poco más tarde, en 2011, se transformó en código abierto y su nombre cambió para Bootstrap. Desde entonces fue actualizado varias veces y ya se encuentra en la versión 4.4. El framework combina CSS y JavaScript para estilizar los elementos de una página HTML.

Es un repositorio de símbolos reutilizables que se utilizan para elaborar nuestras interfaz (botones, widgets, checkboxes, input fields, etc). Este paquete de elementos aporta una rapidez increíble a tu proceso de diseño, ayudándote a poner foco en la solución y no en la construcción visual de elementos, solo tendrás que arrastrar piezas a tu composición y listo.
